왜 Weave인가

기관·전문가에게 꼭 필요한 것만

🗂️

아동별 통합 기록

사진·파일·알림장·메모를 아동별로 체계적으로 기록하고 부모님께 바로 전달해요. 알림장 앱처럼 쉽고, 더 연결되어 있어요.

✨

AI 자동 분석·인사이트

쌓인 기록을 AI가 분석해 놓치기 쉬운 성장 패턴, 주의 신호, 케어팀 간 관찰 일치 여부를 자동으로 알려드려요.

🔗

케어팀 유기적 연결

같은 아동을 담당하는 활동가·복지관·가족과 공개 범위를 설정해 연결하면, 서로의 기록이 유기적으로 통합돼요.

📢

프로그램 검색 노출

기관·전문가 프로그램을 등록하면 Weave를 통해 가족을 찾는 부모님의 검색 결과에 노출돼요. 새로운 연결 기회가 생겨요.

🏛️

기관·전문가 인증 뱃지

기관·전문가 검증 절차를 통해 인증 마크를 받으면 부모님이 더 신뢰하고 연결 요청을 보내요.

👤

담당자 프로필 관리

전문가·기관 내 담당자별 프로필, 전문 분야, 담당 아동을 체계적으로 관리하고 부모님께 투명하게 공개해요.

일정·특이사항을
모두가 함께 확인해요

치료 일정, 활동 매칭, 특이사항을 기관·전문가와 부모님이 함께 저장하고 확인해요. 아이가 아프거나 특별한 일이 있을 때 바로 기록하면 케어팀 모두에게 알림이 가요.

  • 치료·활동 일정 양측 모두 저장·확인
  • 활동가 매칭 후 다음 활동 자동 등록
  • 건강 특이사항 (감기·컨디션 등) 기록
  • 일정 변경·취소 시 케어팀 전체 알림
  • AI가 일정 데이터를 분석해 패턴 인사이트 제공

  • 아동별 성장 지표 자동 추적
  • 케어팀 관찰 일치·불일치 감지

  • 건강 특이사항 반복 패턴 분석
  • 일정·활동 데이터 통합 월별 리포트
  • 주의 필요 아동 자동 알림

케어팀 유기적 연결

같은 아이를 담당하는
모두가 하나로 연결돼요

기관·전문가가 허용한 범위 안에서 활동가·복지관·가족의 기록을 볼 수 있고, 내 기록도 공유해요. 따로따로 기록하던 방식에서 벗어나 아이 중심으로 유기적으로 통합돼요.

  • 아동별 케어팀 관계 지도 (관계맵)
  • 관계자별 공개 범위 세밀 설정
  • 기관·전문가 ↔ 활동가 ↔ 가족 기록 통합 뷰
  • 케어팀 간 메모·알림 공유
  • 부모 확인·피드백 실시간 반영

프로그램 등록 & 검색 노출

기관·전문가의 프로그램을 등록하면
필요한 가족에게 닿아요

부모님이 Weave에서 지역·특성·프로그램 유형으로 검색할 때, 기관·전문가의 프로그램이 검색 결과에 노출돼요. 기관·전문가 인증 마크가 신뢰를 높여줘요.

  • 지역·연령·장애 유형별 맞춤 검색 노출
  • 기관·전문가 인증 마크로 신뢰도 강화
  • 담당자 프로필·자격증 공개
  • 신청 문의 Weave 앱 내에서 바로 연결

기관·전문가 인증
